# ESP32-WeatherWebServer
# 基于ESP32的web气象数据传感网络服务器
基于ESP32，读取传感器数据并在屏幕和网页中使用AJAX动态显示，数据存入数据库中。
支持RESTfulAPI接口，双向控制
